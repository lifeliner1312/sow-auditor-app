ğŸ” TROUBLESHOOTING COMMON ISSUES

Issue 1: "Module not found" error
----------------------------------
Solution:
  - Check files are in correct locations
  - Verify module files renamed correctly (removed prefixes)
  - Run: pip install -r requirements.txt

Issue 2: "DEEPSEEK_API_KEY not found"
--------------------------------------
Solution:
  - Check .env file exists in root directory
  - Verify .env has: DEEPSEEK_API_KEY=sk-...
  - No quotes needed around the key
  - No spaces in the line

Issue 3: OCR not working on scanned PDFs
-----------------------------------------
Solution:
  - Install Tesseract OCR
  - Windows: Add to PATH or set TESSERACT_PATH in .env
  - Test: Run "tesseract --version" in terminal

Issue 4: Email sending fails
-----------------------------
Solution:
  - Use Gmail App Password (not regular password)
  - Enable 2FA on Gmail account first
  - Check port 587 not blocked by firewall
  - Verify EMAIL_SENDER and EMAIL_PASSWORD in .env

Issue 5: "Invalid pillar: Timeline" error
------------------------------------------
Solution:
  - Already fixed in provided files
  - App uses "Schedule" not "Timeline"
  - If error persists, re-download config.py

Issue 6: PDF generation fails
------------------------------
Solution:
  - Check reports/pdf/ directory exists
  - Verify write permissions
  - Run: pip install reportlab --upgrade
